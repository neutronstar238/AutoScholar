# 🚀 AutoScholar

自主学术研究与商业化 AI Agent 系统 - **V1.0.0 正式发布**

## ✨ 功能特性

- 🔍 文献检索 (arXiv)
- 📝 研究笔记生成
- 💡 研究方向推荐
- 🤖 多 Agent 协作
- 📊 数据持久化
- 💬 飞书集成

## 🚀 快速开始

```bash
docker-compose up -d
```

访问：http://localhost:3000

## 📦 已部署功能

- ✅ 前端界面 (Vue 3)
- ✅ 后端 API (FastAPI)
- ✅ 数据库 (PostgreSQL + pgvector)
- ✅ 缓存 (Redis)
- ✅ 飞书集成
